#  DSA Grind Journal

This repository documents my **Data Structures & Algorithms (DSA) learning journey**.

The goal of this repo is to:
- To stay **accountable** during my DSA grind
- To record **what I learn every week**
- To track patterns in mistakes and gaps in understanding
- To improve problem-solving thinking, not memorization
- To look back later and see real progress

This is a **learning journal**, where I experiment but keep on practicing.
I believe writing about learning makes understanding deeper and more permanent.

---

##  What this repository contains

-  Weekly folders with solved problems
-  Notes and explanations (when needed)
-  Revisit problems and optimize solutions
-  Reflections on the week's progress and performance in summary.txt in each folder
-  Monthly progress report and thoughts in the readme.md file
